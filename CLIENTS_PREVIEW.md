```
┌────────────┐
│   Card     │  ← Border: Gray-700
│  Content   │
└────────────┘

┌────────────┐
│   Card     │  ← Border: Green-500/50

│    ↑       │  ← Shadow: Green glow

```

```
User Clicks Card
       ↓

       ↓
Opens Client Website (New Tab)
       ↓
Current Page Remains Open
```

## 💡 Key Visual Features

1. **Glassmorphism Effect**
   - Backdrop blur on cards
   - Semi-transparent backgrounds
   - Subtle borders

2. **Gradient Overlays**
   - Unique gradients per client
   - Opacity transitions on hover
   - Smooth color shifts

3. **Depth & Layering**
   - Cards appear to float
   - Shadows create depth
   - Z-axis transforms

4. **Micro-interactions**
   - Logo scale on hover
   - Arrow translations
   - Border glow effects
